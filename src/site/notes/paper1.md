# 摘要
1. YAG 晶体是一种典型的硬脆材料，莫氏硬度达 8.5，常温下不溶于任何酸碱，加工难度大
2. 基于游离磨料研磨方法，在研磨过程中逐级减小碳化硼磨料粒径。研究了研磨压力、研磨盘、摆轴转速和研磨液中碳化硼质量分数等参数对研磨效果的影响得出最佳研磨参数
3. 通过截面显威法测量出 YAG 晶体研磨后的亚表面损伤的深度，确定后续抛光去除量，并探究了亚表面损失深度和表面粗糙度的关系
4. 当研磨压力为 44.54 KPa、研磨盘和摆轴转速为 60 rpm、研磨液中碳化硼质量分数为 15%时，每个研磨步骤均取得最好的研磨效果
