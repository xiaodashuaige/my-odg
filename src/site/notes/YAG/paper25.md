# 2025-10-20 : paper25
---
## 摘要
1. 为了快速、准确预测微晶玻璃在研磨加工过程中的亚表面损伤状况，制定合理的加工工艺。通过*建立微晶玻璃的离散元模型*，仿真分析了研磨压力等研磨工艺参数对工件亚表面损伤的影响规律，并采用角度抛光法对研磨后微晶玻璃的亚表面损伤状况进行了实验验证
2. 本文建立了微晶玻璃离散元模型，分析了微晶玻璃研磨过程中裂纹和残余应力的形成机制，探索了研磨工艺参数对微晶玻璃亚表面裂纹和残余应力的影响规律

2. 离散元仿真方案设计
	1. 本文固定其他加工参数不变，探究研磨压力对亚表面微裂纹和残余应力的影响规律
	2. 根据不同研磨压力对应的金刚石磨粒切入深度，*使磨粒与工件表面保持对应的切深，恒定磨粒的运动速度，并控制磨粒平行于工件表面运动相同的距离*
	3. 待磨粒完成切削后，退出工件，充分松弛材料模型，最终获得不同研磨压力对应的亚表面微裂纹和残余应力分布情况
	4. 为避免模型在磨粒切削过程中产生位移和晃动、影响仿真结果的准确性，需要在模型右侧和底部添加墙体约束

## 仿真结果分析
1. 亚表面微裂纹![Pasted image 20251021115215.png](/img/user/attachment/Pasted%20image%2020251021115215.png)
	1. 在不同压力下，微晶玻璃材料组织都*或多或少有被去除的现象*，在材料表面*可见明显的脆性断裂*及*断裂后产生的表面凹坑*
	2. 其中当研磨压力为3.5 kPa时，工件亚表面仅存在少量微裂纹，且无明显的纵向裂纹，其最大裂纹深度为1.23 μm；当压力增大到15 kPa时，工件亚表面存在大量的纵向裂纹，最大裂纹深度为2.07 μm
2. 亚表面残余应力![Pasted image 20251021115232.png](/img/user/attachment/Pasted%20image%2020251021115232.png)![Pasted image 20251021115242.png](/img/user/attachment/Pasted%20image%2020251021115242.png)
	1. 工件表面在x方向的残余应力主要为压应力，在y方向主要为拉应力，且随着研磨压力的减小，x方向和y方向的应力均有逐渐减小的趋势
	2. 在裂纹的尖端位置有较大的拉应力
	3. 在研磨加工时，残余应力主要以拉应力和压应力 2种形式表现
	4. *工件表面和内部压应力有利于提升工件疲劳强度和耐磨性，而拉应力更容易使工件产生微裂纹和裂纹扩展*
	5. σxx和σyy均*在工件表面表现为较大的压应力*，并在一定深度后压应力值急剧下降，转变为随着深度增大而增大的拉应力，拉应力值到达顶峰后下降，直至在一定深度后趋于0并保持稳定![Pasted image 20251021115300.png](/img/user/attachment/Pasted%20image%2020251021115300.png)
		1. 以应力值大于10 MPa作为残余应力存在的标准，σyy的分布深度要明显大于σxx的，因此，*以σyy的分布深度作为残余应力层存在的最大深度*
		2. 分析不同研磨压力下的残余应力层深度可得，*随着研磨压力的增大，加工后残余应力层深度增大*
3. 微裂纹数量
	1. 示意图![Pasted image 20251020210025.png](/img/user/attachment/Pasted%20image%2020251020210025.png)
	2. 当研磨压力由3.5 kPa增大到5 kPa时，工件表面微裂纹数量由589增加到了713
		1. 这是由于在压力增大过程中形成了新的微裂纹，从而使裂纹数量有所增加
	3. 当研磨压力继续增大到7 kPa时，微裂纹数量降低为604
		1. 这是由于微裂纹出现了扩展并且相互融合形成了更大的裂纹，因此裂纹数量有所下降
		2. 在压力增大的过程中，微裂纹萌生、扩展以及融合的过程交替进行，从而导致亚表面微裂纹数量呈波浪形变化
	4. 当研磨压力增大到15 kPa时，微裂纹的扩展与融合占据主导地位，使微裂纹数量急剧下降
	5. 当压力增大到一定程度时，微裂纹会不断扩展，最后形成肉眼可见的裂纹，甚至发生断裂
4. 在不同研磨压力下，残余应力层的深度均明显大于微裂纹层深度
	1. 在实际加工过程中，*拉应力是引起裂纹产生和扩展的主要原因*，当尖端的局部拉应力超过极限强度时，材料内部才能产生新的开裂或使原有的裂纹发生进一步延伸扩展，故在裂纹的尖端有应力集中现象，并围绕裂纹尖端形成了应力场
	2. 在深度方向上，随着与尖端距离的增大，应力场内应力逐渐减小，因此*残余应力层的深度必然大于微裂纹层的深度*
5. 亚表面损伤试验验证
	1. 亚表面微裂纹
		1. 偏差由多种原因共同导致，首先在离散元仿真中，将材料简化为二维颗粒的聚合体，*通过颗粒间的连接键来模拟材料的力学性能，与微晶玻璃内部真实的微观结构有一定的区别*
		2. 其次，在仿真中简化了磨粒与工件之间的相对运动，且*忽略了磨粒之间的相互影响*，研磨实验采用的是多磨粒研磨加工，而仿真实验采用了*单磨粒研磨加工*
		3. 最后，忽略了*研磨液和研磨垫表面树脂微凸起*的影响，尤其当研磨垫表面的基体微凸起和研磨液承压较大时，会造成实验结果略小于仿真值
	2. 残余应力
		1. 对于残余应力的检测，本文采用了[[YAG/差动腐蚀法原理\|差动腐蚀法原理]]
			1. 对无表面损伤的基体样件和加工后的工件在相同条件下进行腐蚀，由于*加工后样件（亚）表面组织存在微裂纹以及晶格畸变等缺陷，其腐蚀速率与基体样件的腐蚀速率存在差异*
			2. 根据需求将腐蚀过程设置为n个时间段，单个时间段越小，则试验精度越高，*当第i与i+1 时间段中出现基体样件与加工后样件腐蚀速率差值接近0时*，则*认为在i时间段内加工后工件亚表面损伤消失*，*即残余应力消失*
			3. 再根据腐蚀速率计算对应深度 ![Pasted image 20251020210737.png](/img/user/attachment/Pasted%20image%2020251020210737.png)

## 结论
1. 当利用固结磨料研磨加工微晶玻璃时，加工后存在纵向、横向微裂纹和残余应力等表面和亚表面损伤
2. 研磨压力越大，亚表面微裂纹层和残余应力层深度越大
3. 固结磨料研磨加工后，*残余应力层分布深度大于微裂纹层深度，在微裂纹的尖端存在较大的残余拉应力*
