# 2025-10-19 : paper21
---
## 摘要
1. 本文提出利用离散元法预测评价硫化锌晶体加工后的亚表面微裂纹和残余应力，分析磨粒粒径等工艺参数对亚表面损伤的影响规律，为硫化锌加工工艺的优化提供理论指导
2. 主要研究内容
	1. 建立硫化锌晶体的离散元模型，在PFC2D中根据硫化锌的材料特性和力学性能，建立二维BPM模型，在颗粒间添加连接键并设置初始参数
		1. 利用[[YAG/单轴压缩、巴西拉伸、三点弯曲\|单轴压缩、巴西拉伸、三点弯曲]]来校核模型材料
	2. 利用离散元法探究研磨工艺对硫化锌亚表面损伤的影响规律。*根据磨粒粒径、研磨垫的特性、研抛压力以及硫化锌晶体的力学性能，建立固结磨料研抛的平均切深模型，获得不同粒径对应的切深*。将复杂的加工过程*简化为单颗磨粒以固定切深进行匀速直线运动*，得到模拟切削后的亚表面微裂纹和残余应力分布情况
	3. 分别利用*角度抛光法和差动腐蚀法*检测硫化锌工件的*亚表面微裂纹分布情况和残余应力层深度*，并探究工件表面质量对耐腐蚀性的影响。分析对比离散元仿真结果和实验测量结果，证明离散元法预测评价固结磨料研磨硫化锌后亚表面损伤情况的可行性

## 绪论
1. 课题背景及意义
	1. *红外技术*的飞速发展离不开对红外光学材料的研究和探索，红外光学材料是红外技术的核心，*其红外透过率决定了红外技术的精度，而机械性能则影响红外元器件的使用寿命*
	2. 其中*硫化锌有良好的投射和折射性能，价格便宜且容易加工，断裂强度较高，抵抗环境影响的能力强*，是红外窗口片和整流罩的主要材料
	3. 红外光学器件的性能，在很大程度上受加工工艺的影响，高品质的加工表面是硫化锌器件宽波段高透性的重要保证
	4. 硫化锌属于难加工软脆材料，由于其软、脆的特性，在加工过程中，磨粒及废屑极易嵌入工件表面且加工过程中容易产生裂纹，形成脆性去除，导致加工后的亚表面损伤大，从而影响器件的表面质量和使用性能，给超精密加工带来了极大挑战
	5. 目前，亚表面损伤评价方法，如磁流变斑点法、角度抛光法、差动腐蚀法均是破坏性检测手段，尽管精度较高，但也存在破坏样品、实验工作量大、检测结果准确性受操作经验影响等不足，因此，进行加工前的预测尤为必要

		2. *由于其硬度较低，在加工过程中脱落的磨粒和其他杂质材料极易出现嵌入和拔出，造成严重的损伤*，这些损伤在后续的加工中难以完全消除
		3. 在软脆材料加工过程中*机械去除和磨料摩擦接触共同作用*，且磨料的形态多样，*容易产生摩擦磨损和脱落自锐*，材料的去除机理比较复杂，因此在持续加工过程中，同时保持加工工艺稳定和加工效率也有一定的困难

## 固结磨料研磨硫化锌二维离散元模拟
1. 固结磨料研磨后的亚表面损伤*影响硫化锌器件的使用性能和使用寿命*，而*亚表面损伤的检测往往较为繁琐*。通过*二维离散元模拟切削，可以预测加工后的微裂纹、残余应力等亚表面损伤情况，有利于提高加工效率，促进加工工艺优化*
2. [[YAG/单颗磨粒平均切深模型\|单颗磨粒平均切深模型]]
3. 单颗磨粒模拟切削模型的建立
	1. 在实际研磨过程中，磨粒相对于工件的运动由研磨垫转速、工件转速以及工件安放位置共同决定，为复杂的变速旋转运动，但在PFC中，模拟颗粒的运动距离较短，可以*近似地认为磨粒相对于工件做匀速直线运动*
	2. 本文模拟微切削采用单因素效应法，恒定磨粒的运动速度和研磨压力（20KPa），根据*不同的金刚石磨粒粒径确定磨粒切入的深度，研究切削后的亚表面损伤情况，探究磨粒粒径对硫化锌工件亚表面微裂纹和残余应力分布情况的影响*
	3. 在开始试验后，磨粒沿着X轴作匀速直线运动，设定颗粒的运动距离恒定，*完成切削后，退出切削颗粒，并充分松弛材料模型，得到亚表面微裂纹数量和分布情况以及亚表面残余应力的数值和分布*
	4. 将金刚石磨粒切削刃简化为直角，单颗磨粒模拟切削模型如图3.2所示，磨粒切削刃刃尖和材料表面的相对深度根据计算的单颗磨粒平均切深确定，*材料模型右侧及底部添加墙体进行约束位移，以保证材料模型在微切削模拟试验过程中保持稳定，防止模型产生位移或晃动，以确保模拟切削的准确性*![Pasted image 20251021210612.png](/img/user/attachment/Pasted%20image%2020251021210612.png)
4. 切削后亚表面微裂纹模拟
	1. PFC 中裂纹产生机制
		1. 在 PFC 中，材料模型中的裂纹通过离散裂缝网络(discrete fracture networks，DFNs)来表示
		2. DFN作为断裂的集合，*在二维中表现为颗粒间中段线的集合体，在三维中则为颗粒间的中平面或凸多边形的集合体，离散裂缝集合中的每个元素都是一个断裂事件*，且断裂事件属于有限尺寸的元素，彼此之间相互离散
		3. DFN可以*通过PFC中系统内置的模板生成*，这些模板统计断裂信息，可以描述系统中断裂的分布情况，计算和查询离散的裂缝之间相互的交点并以此为依据分析裂缝的连通性
		4. 在本文使用的二维BPM模型中，颗粒之间*通过接触模型产生的连接键紧密连接在一起*，以对抗外力导致的颗粒间相互运动。追踪记录仿真运行过程中*线性平行连接键的变化行为，当模型中连接键的断裂失效，则该位置产生断裂*

6. 小结
	1. 离散元法仿真中，硫化锌工件表面材料的去除模式属于脆性去除，加工后存在表面凹坑等表面损伤，且固结磨料垫使用的磨粒粒径越大，加工后的表面质量越差
	2. 模拟微切削后硫化锌工件模型的亚表面产生横向微裂纹和纵向微裂纹，微裂纹的浓度随深度的增加逐渐降低。金刚石磨粒的粒径对微裂纹分布的深度和微裂纹的数量有较大的影响， 磨粒的粒径越大，加工后的亚表面微裂纹层深度和微裂纹数量增大，亚表面损伤越严重
	3. 模拟切削后工件内部残余应力𝜎𝑦𝑦的分布深度以及亚表面存在的拉应力明显大于𝜎xx且工件的亚表面残余应力的分布情况受裂纹分布的影响，裂纹的尖端位置附近有较大的残余拉应力
	4. 磨粒粒径对加工后的残余应力影响显著，粒径越大，最大残余拉应力值越大且残余应力层深度越深

## 硫化锌亚表面损伤的检测与评价
1. 实验设计与表征
	1. 步骤
		1. 根据相同的配方（金刚石微粉质量占比、树脂型号、其他添加剂种类及占比均相同），选用不同磨粒粒径的金刚石微粉制作固结磨料研抛垫，并用油石修整研抛垫表面，保持有效研磨表面平整，保证研抛垫凸起部分金刚石磨粒出露
		2. 在平面精密环抛机上利用相同磨粒粒径的研磨垫充分加工硫化锌工件，保证工件表面平整性及试验开始前工件表面的一致性，提高试验结果的可对比性
		3. 分别用5μm、15μm、25μm、30μm磨粒粒径的固结磨料研抛垫加工硫化锌工件，为保证研磨垫表面磨粒相对于工件有足够大的切削速度，工件摆放位置应尽量靠近研磨垫边缘，加工过程中使用0.3％OP和3％三乙醇胺配置的水溶液作为研磨液。加工后对加工后的工件进行编号
		4. 利用 nannomap 三维形貌仪检测加工后硫化锌工件的表面粗糙度𝑅a
	2. 磨粒粒径对工件表面粗糙度的影响
		1. 随着金刚石磨粒粒径的增大，研抛垫表面*出露磨粒数量减少，单个磨粒施加给工件的载荷增大，故磨粒切入工件的深度增加*
		2. 磨粒切削过程中产生的切削力以及材料表面的塑性形变程度增加，应力集中现象更加明显，研抛过程中产生的划痕和凹坑等表面缺陷更加严重，材料表面出现更多的脆性破碎，因此加工后工件的表面粗糙度增大
2. 亚表面微裂纹的表征
	1. [[YAG/亚表面损伤检测方法\|亚表面损伤检测方法]]，补充二的角度抛光法
	2. 角度抛光法中的腐蚀液的选择
		1. 硫化锌在酸性溶液中能被腐蚀![Pasted image 20251021213241.png](/img/user/attachment/Pasted%20image%2020251021213241.png)
		2. 本实验最终选择5mol/L的HCl溶液作为腐蚀液腐蚀4min
	3. 工艺参数对工件亚表面微裂纹深度的影响
		1. 经腐蚀液腐蚀，亚表层原本闭合的微裂纹显现![Pasted image 20251021213529.png](/img/user/attachment/Pasted%20image%2020251021213529.png)![Pasted image 20251021213545.png](/img/user/attachment/Pasted%20image%2020251021213545.png)
		2. (a)为工件表面，可见明显的划痕、微裂纹分布密集，(b)至(e)深度增加，划痕变浅消失，微裂纹浓度逐渐下降，至(f) 中红线位置，深度为6.27μm时，达到微裂纹终点位置，更深处不再出现微裂纹，*此深度即为微裂纹层深度*
		3. 图4.10为粒径5μm，15μm，30μm磨粒加工后的微裂纹终点图片，对比不同粒径加工后微裂纹终点深度可知，*随着磨粒粒径的增大，微裂纹分布的深度明显增加*
3. 亚表面残余应力的表征
	1. [[YAG/差动腐蚀法原理\|差动腐蚀法原理]]
	2. 实验设计
		1. 腐蚀实验开始前利用粒径为5μm、15μm、25μm、30μm磨粒粒径的固结磨料研抛垫分别对加工硫化锌工件两面进行研磨加工，并依次编号为1-4号
		2. 再用磨粒粒径25μm的固结磨料垫通过同样方法加工5号硫化锌工件作为标准样件
		3. 为保证硫化锌工件能与腐蚀液充分接触，并减少腐蚀环境变化引入的误差，且自制支撑架如图4.11 所示，利用该支撑架可以使得腐蚀过程中工件所有表面充分暴露于腐蚀液中，并能同时腐蚀所有1-5号工件，保证所有工件同步放入和取出，减少了腐蚀时间差异带来的误差![Pasted image 20251022113840.png](/img/user/attachment/Pasted%20image%2020251022113840.png)
		4. 差动腐蚀法中的腐蚀速率需要通过工件质量的变化来计算，由于单次腐蚀掉的材料质量很小，因此精确的称量有助于提高差动腐蚀速率计算结果的准确性
			1. 本实验中称量所用电子天平的精度为 0.1 毫克，为尽量减小称量过程中引入的测量误差，每个工件均称量 6 次取并取其平均值作为最终结果
	3. 具体步骤
		1. 选用5mol/L的盐酸溶液作为腐蚀液，利用水浴锅加热至60℃并保持温度恒定
		2. 称量1-5号工件的初始质量后，将工件安放在支撑架上后放入盐酸溶液中，确保腐蚀液完全没过所有工件，腐蚀2min后取出工件并迅速洗去腐蚀液，再用去离子水超声清洗3min并烘干工件，再次称量每个工件的质量，计算腐蚀深度 ![Pasted image 20251022114004.png](/img/user/attachment/Pasted%20image%2020251022114004.png)
		3. 重复1中的步骤，计算每次腐蚀的腐蚀深度和腐蚀加速度，直至腐蚀速度趋向于平缓，腐蚀加速度趋向于0时，工件亚表面损伤层被完全腐蚀，停止腐蚀
		4. 由于试件在研磨*加工前经过充分腐蚀*，因此*以试件稳定后的腐蚀速率作为工件侧面的腐蚀速率*，根据侧面面积占比计算每次腐蚀时侧面的腐蚀量以*排除工件侧面的影响*，计算腐蚀不同时间后的腐蚀深度，根据亚表面损伤层*恰好被完全腐蚀的时间节点获得此时的腐蚀深度*，即为亚表面残余应力层的深度
	4. 工艺参数对工件亚表面残余应力的影响
		1. 在腐蚀过程中，由于工件加工后的表面存在大量的划痕、凹坑等表面缺陷，腐蚀液与工件表面接触面积大，腐蚀的程度剧烈，因此*在腐蚀初期，腐蚀速率很大*，且由于磨粒粒径越大，加工后的表面质量越差，故*腐蚀速率也越大*
		2. 随着腐蚀的进行，表面缺陷被腐蚀掉，且随着深度的增加，*亚表面微裂纹的浓度逐渐减小，腐蚀速减缓*，但在一段时间后，腐蚀速率又有一定的回升，随后继续保持腐蚀速率减小至最终趋于平缓。这是*由于局部的应力集中会加速腐蚀速率*，*残余应力随着深度的增加先增大后减小*，*残余应力与微裂纹共同作用，导致腐蚀速率的曲线呈现先急速下降*，后下降速度放缓，在经过腐蚀速率的回升和下降后最终趋于稳定，此时残余应力层和微裂纹层被完全腐蚀，腐蚀速度稳定并接近于基体的腐蚀速率。因此*腐蚀加速度的第三个零点处视为应力层恰好被完全腐蚀*。
		3. 磨料粒径5μm的固结磨料垫加工后的硫化锌工件在盐酸溶液中耐腐蚀性好，其腐蚀速率始终小于基体的腐蚀速率，无法用差动腐蚀法测量取亚表面损伤层深度
	5. 工件表面质量对耐腐蚀性的影响
		1. 经过腐蚀，*工件表面粗糙度均有所上升*，且*磨粒粒径越大，表面粗糙度上升程度越大*，磨粒粒径为5μm时，经过腐蚀，工件仍有较好的表面质量
		2. 腐蚀前，1号工件表面非常光滑，有*较好的镜面效果和透光性*，表面可见少量划痕，2-4号工件表面*存在大量的划痕和表面缺陷，镜面效果和透光性差*，表面光滑程度随磨粒粒径的增大而减小
		3. 腐蚀后，2-4号工件表面呈现磨砂质感，光滑度和透光性较腐蚀前有所下降。而1号工件表面镜面效果下降但*仍有较好的透光性*，*表面划痕被严重腐蚀*，划痕清晰可见且*腐蚀后划痕加粗，划痕密集处腐蚀程度严重*
		4. 在磨粒粒径较小时，固结磨料加工后的*工件表面缺陷少，工件表面微裂纹浓度低*，腐蚀时，腐蚀液腐蚀划痕凹坑和微裂纹等表面缺陷，而*没有表面缺陷的地方耐腐蚀性能好，腐蚀速率缓慢*
		5. 磨粒粒径较大时，*加工后工件表面质量差，各类表面缺陷分布密集*，在腐蚀液作用下，这些*表面缺陷被严重腐蚀，腐蚀速率增大，表面质量进一步下降*
		6. 优良的加工表面能有效提高硫化锌工件的耐腐蚀性，在加工过程中应尽可能减少表面和亚表面的缺陷
4. 亚表面损伤离散元法预测结果与实验结果对比
	1. 不同磨粒粒径加工后差动腐蚀法测得的*残余应力层深度均明显大于角度抛光法测得的微裂纹层深度*
		1. 拉应力是引起裂纹产生和扩展的的主要原因，当尖端的局部拉应力超过极限强度时，材料内部才能产生新的开裂或原有的裂纹发生进一步延伸扩展
		2. 故在裂纹的尖端有应力集中现象，并*围绕裂纹尖端形成应力场*，在深度方向上，*随着与尖端距离的增大，应力场内应力逐渐减小*，因此残余应力层的深度必然大于微裂纹层的深度
	2. 在磨粒粒径为 5 μm 时，实测值小于仿真值，而在粒径大于 5 μm 时，实测值均大于仿真值
			1. 实际加工中存在粒径大于仿真所设平均粒径的磨粒，造成实际微裂纹层深度大于仿真结果
			2. 在粒径较大时，磨粒的出露更高，加工过程中基体与工件表面接触少，因此磨粒承受绝大部分的加工载荷，*基体受力可忽略不计*，而在*磨粒粒径较小时*，*磨粒出露高度小，基体与工件的接触面积增加，所分摊的加工载荷增大*，*磨粒实际切入工件深度小于理论计算的平均切深*，因此，粒径为 5 μm 时实测微裂纹层深度小于仿真值
	3. 预测与实际结果的偏差由多种因素共同导致
		1. 离散元仿真中，将三维立体的材料简化成了二维颗粒的聚集体，材料模型的结构、颗粒的排布方式以及粒径范围均与实际材料内部的微观结构有很大的区别
		2. 本文建立的硫化锌材料模型仅校核了杨氏模量、泊松比、拉伸强度和弯曲强度，模型材料的其他机械性可能与实际材料存在一定的偏差，这将在一点程度上影响仿真结果的准确性
		3. 离散元仿真通过颗粒间线性平行连接键的断裂来模拟微裂纹的产生，将材料模拟加工过程中的受力和碎裂情况理想化，实际材料内部裂纹和应力的产生有着更复杂的力学行为
		4. 在实际加工过程，由于固结磨料垫和工件的转动，磨粒将呈现具有半确定性的复杂曲线运动，而在本文的离散元仿真中，将其简化为匀速直线运动
		5. 本文采用单颗磨粒模拟切削，忽略了加工过程中磨粒之间的相互影响以及磨粒的磨钝，脱落、嵌入和拔出等行为对表面和亚表面损伤的影响
		6. 差动腐蚀实验中，腐蚀环境的影响和测量精度带来的误差也难以避免

## 展望
1. 受限于计算机的计算能力，本文将硫化锌材料简化为二维的BPM模型，而在实际加工中，材料内部的亚表面损伤形态是三维的，可以*采用PFC3D建立硫化锌的三维BPM模型*， 进行模拟切削，不仅可以得到微裂纹和残余应力在沿着深度和切削方向的分布情况，还可以获得磨粒切削路径两侧的亚表面损伤情况
2. 在本文中，固结磨料研磨加工硫化锌简化为单颗磨粒模拟切削，忽略了磨粒之间的相互影响，在接下来的探索中，可以*模拟多个磨粒同时切削，探究多颗磨粒研磨后的亚表面损伤分布*
3. 硫化锌固结磨料研磨后的亚表面损伤受磨粒的粒径浓度、固结磨料垫结合剂性质、研抛压力、研抛垫和工件转速、研抛液等多种因素的影响，本文*仅考虑了其中影响较大的磨粒粒径对亚表面损伤的影响，仍需进一步考虑其他因素的影响*
