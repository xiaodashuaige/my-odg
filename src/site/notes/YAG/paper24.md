# 2025-10-20 : paper24

## 摘要
1. 为更好地了解固结磨料研磨垫表面形态，尤其是研磨垫中的*金刚石、孔隙、金刚石脱落坑等的分布特征*，提出一种基于深度学习的固结磨料研磨垫表面形态分析方法
2. 利用徕卡DVM6数字显微镜及其配套软件获取固结磨料研磨垫表面图像；然后，采用python3+OpenCV对图像进行预处理，并利用标注软件Labelme对图像进行标注，用于后续的训练和测试；最后，运用深度学习框架Tensorflow搭建Mask R-CNN模型
3. 固结磨料研磨垫（fixed abrasive pad，FAP）是*将微米级金刚石磨粒均匀分散到高聚物基体中，经过固化成形制备出的一种含磨粒的研磨垫*
4. FAP表面主要由金刚石、孔隙、金刚石脱落坑以及高聚物基体组成。其中，*高聚物基体作为FAP的主要成分，其硬度对FAP的加工性能有着重要影响*
5. 霍凤伟等提出了一种*识别细粒度金刚石砂轮表面磨粒分布*的方法，该方法依据*磨粒轮廓频率特征、磨粒间距和磨粒曲率半径*识别砂轮表面金刚石磨粒
6. 贾坡等提出了一种基于机器视觉的*滚抛磨块缺陷检测方法*，利用图像灰度化、阈值分割、形态学处理提取磨块区域和黑心缺陷区*域*，该方法能够对烧结型球状磨块的圆度和黑心缺陷进行数字化检测
7. 赵玉康等提出的*金刚石线锯全表面图像磨粒分布密度检测方法*能提高其磨粒分布密度在线检测精度，相较于单相机检测结果，误差大大降低
8. 为了更好地了解FAP表面形态，*对FAP表面的金刚石、孔隙以及金刚石脱落坑进行目标检测和分割*。目标检测一直是计算机视觉领域研究的热点
	1. 主要任务是完成目标的分类及定位
	2. 传统的目标检测方法
		1. 首先，在图像上选择一些候选区
		2. 然后，人工对这些候选区进行特征提取
		3. 最后，送入分类器进行分类
		4. 其中，人工提取特征的质量直接影响检测的准确性。由于实际图像受干扰较多，而且一旦目标特征比较抽象，背景较为复杂，就不能很好地提取目标特征，所以人工特征提取往往存在一定局限性
		5. 而基于深度学习的目标检测算法则是让网络通过训练自己提取目标特征，利用庞大的数据量使算法具有较高的泛化能力
	3. 考虑到固结磨料研磨垫*表面图像背景较为复杂*，所需检测的目标，尤其是*孔隙以及金刚石脱落坑的特征较为抽象*，采用传统的目标检测算法很难有效地对目标进行识别
		1. 因此，采用深*度学习模型Mask R-CNN* 对固结磨料垫表面图像中的金刚石、孔隙以及金刚石脱落坑进行检测和分割，辅助研究人员正确预测固结磨料研磨垫的加工性能

## Mask R-CNN 模型
1. 总体结构
	1. Mask R-CNN 网络总体结构如图 1 所示，其是一个二阶段目标检测网络，总体分为 2 个阶段![Pasted image 20251020112740.png](/img/user/attachment/Pasted%20image%2020251020112740.png)
	2. 第1阶段，利用特征提取层对图像进行特征提取，特征提取层的主干网络采用的是残差网络（简称“ResNet”）。提取特征之后，结合特征金字塔网络（ feature pyramid net-work，FPN）生成多尺度特征图。得到多尺度特征图之后，候选区域生成网络（region proposal network，RPN） 生成目标图像的候选区域，并将候选区域映射到相应的特征图上
	3. 第2阶段，将得到的候选区域映射到特征图后传入ROIAlign层，*ROIAlign层将得到的候选区域进行下采样得到固定维数的特征向量，再将特征向量送入全连接层进行分类和检测边框回归*。同时，模型在ROIAlign层后面连接了一个用于生成目标掩码的掩码分支
2. ResNet
	1. 理论上，神经网络的性能与其层数是成正比的；但实际上，层数过多的网络*会出现网络退化和梯度消失的问题*，效果反而不如较浅的网络。*ResNet的提出就是为了解决深层网络的网络退化和梯度消失的问题*
	2. ResNet的基本思想是：随着网络的层数增加，网络的性能是先提升后下降，那么一个网络一定存在一个最优的层数，但并不知道最优的层数是多少，因此所设计的深层次网络有很多网络为冗余层。希望这些冗余层至少能够完成恒等映射，这样就可以保证模型性能不会因为这些冗余层而下降。至于哪些层属于冗余层，需要网络训练时学习及判断
	3. ResNet的核心是残差模块，其结构如图2所示。通过加入恒等映射，残差块变得更加容易优化，从而在一定程度上解决了深层网络退化和梯度消失的问题 ![Pasted image 20251020113327.png](/img/user/attachment/Pasted%20image%2020251020113327.png)
3. 特征金字塔网络
	1. FPN的提出是为了解决目标检测中检测多尺度目标时检测精度不足的问题。采用FPN可以有效处理检测过程中多尺度变化的问题![Pasted image 20251020113741.png](/img/user/attachment/Pasted%20image%2020251020113741.png)
	2. *自底向上连接*就是简单的特征提取过程，将ResNet作为骨架，得到不同尺度的特征图C1，C2，C3，C4，C5
	3. *自顶向下连接*是从最高层开始上采样得到高分辨率的特征图M5， M4， M3， M2
	4. *侧向连接*就是将上采样的结果和与之相对的自底向上生成的大小相同的特征图进行融合，得到P2，P3， P4，P5
		1. 此外，M5还经过下采样生成P6（P6在图3中并未画出），P6只会送入候选区域生成网络提取候选框，并不会在P6上切出感兴趣区域送入后续网络
4. 候选区域生成网络
	1. 候选区域生成网络（RPN）是基于 FPN 得到的特征图 P 2，P 3，P 4，P 5，P 6，*在特征图上生成大小、比例不同的锚框*
	2. 首先，RPN 会对生成的锚框进行修正；然后，根据前后景置信度，采用非极大值抑制的方法筛选出较为精准的锚框作为候选区域
5. ROIAlign
	1. 由于全连接层的输入向量维数是固定的，所以在获取候选区域之后，需要将其转化为固定维度的特征向量，才能送入后续的全连接层进行分类和边界框回归
	2. *在Mask R-CNN之前，这一步都由传统的池化算法完成，其过程需要进行2次量化*，包括*将浮点数的候选区域边界量化为整数坐标值*，以及*将量化后的候选区域边界分割成k×k个单元，再对每一个单元进行量化*
		1. 2次量化误差造成特征图上候选区与原图不能准确对应，影响了分割的准确性
	3. ROIAlign则是采用双线性插值的方法获得浮点数坐标值，取消其量化过程，从而大大减小量化误差，极大地提高了小目标分割的准确性

## 固结磨料研磨垫表面形态识别试验
1. 数据集
	1. 要得到泛化能力较强的FAP表面图像检测模型，就要求模型训练所需的*图像数据不能太过相似*
	2. 为得到表面特征差异较大的FAP，制*作3张磨粒粒径、不同成孔剂占比不同的FAP，选择碳酸氢钠作为成孔剂，其占比决定了研磨垫中孔隙的多少*
	3. FAP制备过程：配置合适比例的磨料与高聚物基体以及成孔剂混合物，经机械搅拌均匀后，注入模具热固成形，最后进行固化
	4. 新制备的FAP表面的金刚石被高聚物包裹，*为了使金刚石出露，并形成一定数量的金刚石脱落坑，在对固结磨料垫上的凸起进行图像采集之前，需要对FAP进行修盘处理*
		1. 修盘的具体操作如下：将SiC粉末均匀撒在FAP上，然后将油石压在研磨垫上，反复打磨10 min，再清洁磨盘
	5. 修盘完成后，利用显微镜观察FAP表面，确保FAP表面的金刚石已经出露后，*将每个FAP均匀划分为4个区域。在每个区域上选取5个凸起，3张垫共得到60个凸起*
		1. 利用徕卡显微设备对每个凸起进行图像采集，图像放大后得到 FAP 表面原始图像共 160张
		2. 得到原始研磨垫表面图像后，利用 OpenCV+Py thon 3 对其进行镜像翻转、旋转、调节亮度与对比度、平移等操作，然后进行预处理和扩充，扩充后，共得到得到500 张 FAP 表面图像
	6. 利用Labelme标注工具标注图像中的目标类别，其中的*研磨垫表面的金刚石亮度较高，有明显金属光泽、折线轮廓*；*孔隙亮度较低，有圆弧轮廓*；*金刚石脱落坑亮度较低，有折线轮廓，尺寸相较于孔隙较大*
	7. 利用Labelme，将标注好的500张研磨垫表面图像制作成可用于训练和测试的数据集。*并随机抽取图像的20%作为测试集，用来评估模型的性能*。*剩下80%的图像作为训练集，用来训练模型*。*训练集中随机抽取图像的25%作为验证集，用于在训练过程中调整模型的参数*
2. 模型训练
	1. 利用深度学习框架TensorFlow搭建Mask R-CNN模型
		1. Mask R-CNN是一个庞大的模型，拥有大量的参数，*若采用常规的方法初始化模型，需花费大量的时间训练模型*
		2. MS COCO数据集是一个大规模的数据集，拥有海量的数据。*Mask R-CNN模型经过MS COCO数据集训练后，所得到的模型参数是比较通用的初始化模型时所采用的参数*。采用*迁移学习*的方法将其用于模型的初始化，既节省大量的时间，又能得到一个较好的结果
	2. 试验所采用的优化器是带学习动量的随机梯度下降优化器，为了适应FAP表面形态的识别任务，相比于原模型，采用了更小的锚框尺寸以减少RPN生成的候选区个数
	3. Mask R-CNN的损失函数值由3个部分组成：*分类误差、检测误差、分割误差*，总的损失函数值是这三者之和
3. 评价指标
	1. 在目标检测任务中，为找到所有正样本的能力，常用查全率ξRe评价模型；为预测准确的程度，用查准率 ξPr评价模型 ![Pasted image 20251020115212.png](/img/user/attachment/Pasted%20image%2020251020115212.png)
	2. ξTP即真正例，为模型和人工标注均提取到的目标数；ξFP即假正例，为模型能提取到而人工未进行标注的目标数；ξFN即假负例，为进行人工标注而模型未能提取到的目标数
	3. 查全率和查准率越高，模型的性能越好
		1. 为了两者兼顾，可以通过模型的PR曲线来评价模型的综合性能
		2. PR曲线是通过改变置信度阈值，查准率随着查全率变化的曲线，PR曲线所包围的面积为精度ξAP，而*所有类别的平均精度即ξmAP值反映了模型的综合性能，是目标检测模型中最重要的评价指标*
	4. 试验建立的FAP 表面表征模型以模型在测试集上的ξmAP为评价指标
